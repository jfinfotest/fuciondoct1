## 🚀 Componentes de Landing
Componentes diseñados para páginas principales y secciones de alto impacto.
- **[Hero](./01-landing/hero)**: Secciones de cabecera con acciones y gradientes.
- **[Bento Grid](./01-landing/cuadricula-bento)**: Cuadrículas de características estilo moderno.
- **[Glow Cards](./01-landing/glow-cards)**: Tarjetas con seguimiento de mouse y resplandor.
- **[Roadmap](./01-landing/roadmap)**: Línea de tiempo de progreso del producto.

## 🧱 Estructura y Navegación
Componentes para organizar el flujo de información.
- **[Accordion](./02-estructura/acordeon)**: Secciones desplegables.
- **[Carousel](./02-estructura/carrusel)**: Deslizadores de contenido y multimedia.
- **[Timeline](./02-estructura/linea-de-tiempo)**: Secuencias cronológicas.
- **[Steps](./02-estructura/pasos)**: Guías paso a paso numeradas.

## 💻 Código y Datos
Herramientas para documentación técnica y de desarrollo.
- **[Bloques de Código](./03-codigo-y-datos/bloques-codigo)**: Resaltado de sintaxis avanzado y pestañas.
- **[Terminal](./03-codigo-y-datos/terminal)**: Simulador de línea de comandos interactivo.
- **[Code Explainer](./03-codigo-y-datos/explicador-codigo)**: Tutoriales de código línea por línea.
- [Code Embed](./03-codigo-y-datos/embebidos): Integración con Sandboxes externos.
- [Calendar](./03-codigo-y-datos/calendario): Gestión de eventos y cronogramas.
- [Property Table](./03-codigo-y-datos/tabla-de-propiedades): Tablas de tipos y parámetros.

## 👁️ Visualización
Componentes para mostrar assets y estructuras.
- [File Tree](./04-visualizacion/arbol-de-archivos): Visualizador de estructura de carpetas.
- [Mermaid Diagrams](./04-visualizacion/mermaid): Diagramas generados por texto.
- [Github Repo](./04-visualizacion/github-repo): Tarjetas de repositorio dinámicas.
- **[Media](./04-visualizacion/multimedia)**: Soporte para Video y Audio.
- **[Zoom Image](./04-visualizacion/imagen-zoom)**: Imágenes interactivas a pantalla completa.

## 🛠️ Utilidades
Componentes pequeños para mejorar el detalle y la navegación.
- **[Alertas](./05-utilidades/alertas)**: Notas, advertencias y consejos.
- **[Badges](./05-utilidades/badges)**: Etiquetas de estado.
- **[Kbd](./05-utilidades/kbd)**: Atajos de teclado estilizados.
- **[Tooltip](./05-utilidades/tooltip)**: Información contextual en hover.
