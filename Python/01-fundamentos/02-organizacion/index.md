# Organización y Arquitectura

Esta sección profundiza en cómo FusionDoc estructura la información y cómo el motor interno procesa tus archivos para convertirlos en una experiencia interactiva.

Aprenderás sobre:

- **Estructura de Directorios:** La anatomía del proyecto y el rol de cada carpeta.
- **Motor de Contenido:** Detalles técnicos sobre el procesamiento MDX, la caché por SHA y el enrutamiento dinámico.
- **Niveles de Navegación:** Cómo usar Tópicos, Categorías y Páginas para organizar contenido complejo.
