# Sistema de Íconos

FusionDoc integra **Iconify** nativamente, permitiéndote acceder a más de 200,000 íconos de diversas colecciones sin afectar el rendimiento de carga inicial.

En esta sección aprenderás:

- **Configuración:** Cómo se inyectan los íconos asíncronos.
- **Uso en MDX:** Cómo usar el componente `<Icon />` en medio de tu contenido.
- **Componentes:** Integración con `CodeTabs`, `BentoGrid` y otros elementos del sistema.

Explora la guía detallada a continuación para comenzar a enriquecer visualmente tu documentación.
