# Desarrollo de Aplicaciones Móviles Android

Bienvenido al curso de **Fundamentos de Kotlin**. Este curso está diseñado para llevarte desde los conceptos básicos de sintaxis hasta patrones avanzados de programación funcional, preparando el terreno para el desarrollo profesional de aplicaciones Android.

## 🚀 Objetivo del Curso
Al finalizar estas 4 sesiones, serás capaz de escribir código Kotlin idiomático, seguro y eficiente, aprovechando las características modernas del lenguaje que lo hacen la opción preferida por Google para Android.

## 📅 Plan de Estudios

### [Módulo 1: Fundamentos de Kotlin](./01-contenido)
Explora las 4 sesiones principales:
1. **El Salto a Kotlin**: Sintaxis y Seguridad (Null Safety).
2. **Lógica y Datos**: Funciones y Estructuras de Datos.
3. **POO Moderna**: Data Classes, Objetos y Herencia.
4. **Estilo y Eficiencia**: Lambdas, Scope Functions y Corrutinas.

> [!TIP]
