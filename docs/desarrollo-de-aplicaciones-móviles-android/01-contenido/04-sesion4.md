# Sesión 4: Programación Funcional y Scope Functions

El "estilo Kotlin". Aprender a escribir código declarativo y elegante.

## Temas a Tratar

* **Lambdas:** Sintaxis de funciones de orden superior y el uso de `it`.
* **Operadores Funcionales:** `filter`, `map`, `flatMap` y `reduce` para transformar datos.
* **Scope Functions:** Cuándo usar `let`, `apply`, `run`, `also` y `with` para limpiar el código.
* **Introducción a Corrutinas:** Qué son las `suspend functions` y cómo se maneja la asincronía básica.

---

## 🏆 Proyecto Final
**Objetivo:** Integrar todo en una pequeña aplicación de consola que procese una lista de datos (objetos), los filtre, los transforme y los muestre de forma segura.

> [!CAUTION]
> Aunque las Corrutinas son potentes, requieren un cambio de mentalidad hacia la programación asíncrona. En esta sesión sentaremos las bases conceptuales.
